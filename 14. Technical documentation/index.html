Java

Java es uno de los lenguajes de programación más populares del mundo.

Al ser un lenguaje de programación tan popular, sus características son muy conocidas:

  • Lenguaje fuertemente tipado.
  • Orientado a objetos.
  • Es un lenguaje interpretado.
  • Sensible a mayúsculas.
  • Portable.
  • Robusto.
  • Seguro.

    • Además de ello, es conocido por ser un lenguaje de programación orientado a objetos, en el que cada elemento es considerado un objeto.

    Con Java, es posible desarrollar desde aplicaciones básicas para diversos dispositivos hasta grandes aplicaciones empresariales, ya que su lema es "Write Once Run Everywhere", siendo posible su ejecución en todo dispositivo en el cual cuenta con la Java Virtual Machine instalada.

    Hello World

    Es cierto que Java es un lenguaje verboso, lo que no pone fácil crear el primer Hello World:

    public class HelloWorld {

    public static void main(String[] args) {

    System.out.println("Hello World");


    Desglose del código

    public class HelloWorld declara la clase HelloWorld.

    public static void main(String[] args) será el método principal, necesario para toda aplicación Java.

    System.out.println("Hello, World!"); imprime el texto en consola, siendo System.out el flujo de salida estándar, y System.out.println("Hello World!"); el método que imprime la línea de texto indicado.

    Datos primitivos

    Al ser Java un lenguaje fuertemente tipado, es necesario definir el tipo de dato al definir la variable, por lo que es necesario conocer los tipos de datos primitivos con los que cuenta el lenguaje:

    Byte

    Tipo de dato numérico entero cuyo tamaño son 8 bits.

    Short

    Tipo de dato numérico entero cuyo tamaño son 16 bits.

    Int

    Tipo de dato numérico entero cuyo tamaño son 32 bits.

    Long

    Tipo de dato numérico entero cuyo tamaño son 64 bits.

    Float

    Tipo de dato numérico decimal cuyo tamaño son 32 bits.

    Double

    Tipo de dato numérico decimal cuyo tamaño son 64 bits.

    Boolean

    Su tamaño es 1 bit.

    Define tipos de datos booleanos, únicamente con los valores true o false.

    Son utilizados para controlar el flujo de un programa, a través de condiciones simples.

    Char

    Su tamaño son 16 bits.

    Representa un carácter Unicode.

    String

    String en Java se considera una clase que representa una cadena de texto, una secuencia de caracteres alfanuméricos, además de puntuación y espacios en blanco.

    Todo el contenido de un string debe aparecer entre comillas dobles. Por ejemplo:

    "Hello World!".

    Un String sería el Hola Mundo que imprimiría un System.out.println("Mi Cadena de Texto");.

    La clase String proporciona diversos métodos, útiles para manipular y trabajar con estos elementos.

    Entre los métodos más comunes para trabajar con String, se encuentran:

    length(), que devuelve la longitud de la cadena.

    charAt(int index), que devuelve el carácter en una posición específica.

    toLowerCase(), el cual convierte todos los caracteres de la cadena a minúsculas.

    toUpperCase(), al contrario que el anterior, convierte todos los caracteres de la cadena a mayúsculas.

    Para comparar cadenas, no se debe usar el operador ==, sino el método equals() o equalsIgnoreCase() si no se quiere tener en cuenta las mayúsculas o minúsculas.

    Variables

    Una variable en Java es un espacio de memoria reservada en memoria, en el que se almacena un valor

    Las variables, además de contar con un tipo de dato el cual define el tipo de información que contiene, siempre cuentan con un nombre asignado.

    Ejemplos de declaración de variables, serían:

    int numero = 3;

    long numeroGrande = 27;

    float decimal = 3.14;

    boolean night = true;

    String firstProgram = "Hello World";

    La convención es que las variables se escriban con la primera letra minúscula, y de ser dos o más palabras la primera comienza en minúscula y el resto con la primera letra mayúscula, todas sin espacio.

    Al nombrar una variable, es necesario tener en cuenta que al nombrarlas no se pueden utilizar palabras reservadas del propio lenguaje.

    Constantes

    Una constante en Java es un valor inmutable, ya que una vez se le asigne su valor, no se podrá cambiar.

    Se utilizan para garantizar que datos los específicos que nunca varían, como lo pueden ser el número de los días de la semana o el número pi, se mantengan constantes a lo largo del programa.

    La declaración de una constante en Java se realiza usando las palabras clave static y final.

    Un ejemplo básico de declaración de constante sería:

    static final TIPO NOMBRE_CONSTANTE = valor;

    Donde TIPO es el tipo de dato, como int, double o String.

    Al declarar una constante, siempre se pone la palabra reservada final antes del nombre de la constante para indicar que su valor no puede cambiar.

    La convención para nombrar constantes es escribir los nombres en mayúsculas y, si contienen más de una palabra, separarlas con guiones bajos, por ejemplo: NUM_DIAS_SEMANA.

    Utilizar constantes ayuda a evitar errores, ya que, al ser inmutables, no se pueden modificar accidentalmente durante la ejecución del programa.

    Además, las constantes mejoran la legibilidad del código, ya que su nombre en mayúsculas indica claramente que se trata de un valor fijo y definido previamente.

    También facilitan el mantenimiento del código, porque si se necesita cambiar el valor de una constante, se hace en un solo lugar, y el cambio se refleja en todo el programa.

    Estructuras de control

    Las estructuras de control de flujo son elementos clave en la programación, ya que, gracias a ellos, es posible tomar decisiones y repetir acciones.

    Java cuenta, además de break y continue, con dos tipos de estructuras de control, siendo estas las estructuras condicionales y las estructuras de repetición o bucles.

    Break y continue

    Dichas estructuras, permiten romper o continuar con el flujo del programa.

    Break

    Normalmente, un bucle se ejecutaría hasta su finalización, pero con break se interrumpe y se fuerza la salida del bloque.

    En caso de querer romper esa ejecución en algún punto concreto, es posible utilizar break, permitiendo salir del bloque de sentencias.

    Continue

    Continue, al contrario que break, se utiliza para desplazarse directamente al siguiente bloque, continuando el flujo del programa.

    Sentencias de decisión

    Estas sentencias de decisión, también conocidas como condicionales, son el if, if-else y switch.

    Permiten evaluar una o varias condiciones para decidir qué bloque de código se ejecutará.

    If

    If permite ejecutar un bloque de código si se cumple una condición determinada. De no cumplirse, omitirá dicho bloque y continuará con el siguiente.

    Suele ejecutarse acompañado de else, a pesar de poder ser utilizado de manera individual.

    If-else

    Un if puede incluir también una cláusula else, dando ejecución a otro bloque de código en caso de no cumplirse la primera condición.

    if (condición) {

    // Bloque a ejecutar si se cumple la condición anterior.

    } else {

    // Bloque else, el cual se ejecutará al no cumplirse la condición anterior.

    }

    Switch

    Switch es la estructura adecuada para evaluar múltiples decisiones y ejecutar el bloque seleccionado.

    Es útil para seleccionar una opción entre múltiples casos.

    switch (option) {

    case option1:

    // Código a ejecutar si es option1.

    break;

    case option2:

    // Código a ejecutar si es option2.

    break;

    default:

    // Código a ejecutar si es otra opción.

    }

    Sentencias de repetición o bucles

    Los bucles permiten ejecutar un bloque de código un número, determinado o indeterminado de ocasiones, hasta darse una condición.

    Una vez se cumpla la condición, el programa saldrá de dicho bucle.

    Estas sentencias de repetición son el for, while y do-while.

    For

    For ejecuta un bloque un número determinado de veces.

    Para ello, al declarar un for, es necesario indicar el punto de inicio, la condición para su fin y su incremento o decremento.

    int i;

    for (i = 0; i < 10; i++) {

    // Acción que se repetirá

    System.out.println("Total de repeticiones del for: " + i);

    }

    While

    El bucle while ejecuta un bloque mientras se cumpla una condición. Esta condición, será lo que indique el número determinado de veces que se repetirá la condición.

    Antes de ejecutar por primera vez el bloque, evaluará si se cumple la condición. Una vez realice dicho bloque, volverá a realizar la evaluación, hasta salir del bucle en el momento que deje de cumplirse.

    Por este motivo, es necesario que dentro del bucle while, se encuentre al menos una sentencia que modifique dicha evaluación.

    int i=0;

    while (i <= 10) {

    //Acción que se repetirá.

    i++;

    }

    System.out.println("Total de repeticiones del while: " + i);

    Do-while

    El bucle do-while, desde el principio, ejecuta el bloque de código. Tras ello, verifica si se cumple o no la condición para continuar dentro del bucle o salir de él.

    Con esto, se consigue que el bloque de sentencias se ejecute al menos una vez.

    int i;

    do {

    //Acción que se repetirá.

    i++;

    } while (i < 10);

    System.out.println("Total de repeticiones del do-while: " + i);
